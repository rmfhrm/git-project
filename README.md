# 프로젝트 명:  랜덤 자리 배치 서비스

개발 환경 

* language: HTML, JAVASCRIPT
* OS: WINDOWS

git-master: 선승정

git-branchs: 문준현, 최정재, 한동환

# 주요 기능

* 사용자가 인원을 선택하면 랜덤으로 좌석이 배정됨
* 사람들의 이름이 적힌 엑셀 파일을 올리면 이를 이용하여 좌석을 배정할 수 있음(예정)
* 이전 짝궁과 앉지 않기(예정)
